# 什么是梯度？
原函数在给定的点函数值变化（默认上升）最快的方向，梯度是一个向量。

# 什么是梯度下降？
梯度下降就是 梯度的概念加上一个条件：下降最快。
因此梯度下降就是：原函数在给定的点函数值下降最快的方向。

# 梯度下降的作用？
梯度下降是为了优化损失函数。
